To provide better services, this app requires the following permissions and features:

Device management functions:
1. Enable/Disable Wi-Fi
2. Enable/Disable Wi-Fi hotspots
3. Enable/Disable USB debugging and data transmission
4. Enable/Disable storage access (microSD card)
5. Enable/Disable NFC
6. Enable/Disable data connectivity
7. Enable/Disable the drop-down menu from the status bar
8. Enable/Disable calling
9. Enable/Disable SMS messaging
10. Restart and power-off
11. Obtain root status
12. End the current call
13. Install/Uninstall an app silently
14. Delete app data
15. Stop an app process
16. Enable/Disable installation of apps from a specific app store. When this function is enabled, apps from other app stores or sources (such as ADB mode or a memory card) cannot be installed.
17. Keep a specific app always running
18. Prevent a specific app from running
19. Only allow the installation of a specific app
20. Prevent a specific app from being uninstalled
21. Configure Exchange settings for Huawei emails
